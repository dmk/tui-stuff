# Weather TUI

A terminal weather app using the Open-Meteo API, built with tui-dispatch.

<img src="../assets/screens/weather.png" width="800" alt="Weather TUI">

_This is a creative sketch only, not ready for use._

---

## Features

- Real-time weather data from [Open-Meteo](https://open-meteo.com/)
- City search with geocoding
- Auto-refresh at configurable intervals

## Controls

- `s`: Open city search
- `q`: Quit

### Search Mode

- Type to search cities
- `↑`/`↓`: Navigate results
- `Enter`: Select city
- `Esc`: Cancel
